# 예제

```kotlin
data class Player(val name: String)

//...

val commandDispatcher = CommandDispatcher<Player> {
    //  "할복" 명령어를 만들어 보고 싶으세요?

}

commandDispatcher.execute("할복", Player("hong gil dong"))
```
